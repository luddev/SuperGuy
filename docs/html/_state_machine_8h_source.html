7 /* FINITE STATE MACHINE
8  * The machine is in only one state at a time;
9  * the state it is in at any given time is called the current state.
10  * It can change from one state to another when initiated by a triggering event or condition;
11  * this is called a transition.
12  * A particular FSM is defined by a list of its states,
13  * and the triggering condition for each transition.
14  */
